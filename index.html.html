현장의 도메인 지식을 이해하고, 노이즈 가득한 실전 데이터 속에서 의미 있는 가치를 정제하여 기업의 수율 개선을 이끄는 신소재 출신 데이터 엔지니어입니다.

현장의 데이터는 왜 버려지는가?

제조 현장의 원천 데이터 유실 실태

도메인을 모르는 상태에서 수집되는 데이터는 노이즈가 심해 80% 이상이 정제되지 못하고 버려집니다. 저는 이 문제를 공학적 직관으로 해결합니다.

"현장을 모르면 가짜 데이터를 고른다"

실제 공정 레일이나 실험실에서 발생하는 온도, 압력, 격자 구조 변수는 가혹한 물리적 환경에 노출되어 있습니다.

저는 신소재공학을 전공하며 물질의 메커니즘을 직접 다뤄봤기에, 어떤 정형/비정형 데이터가 수율 개선에 직결되는 '진짜 신호(Signal)'인지 정밀하게 판별해 낼 수 있습니다.

데이터 파이프라인 구축 프로세스

현장의 원천 데이터를 수집하여 실제 비즈니스 가치로 변환하기 위한 3단계 자동화 아키텍처입니다.

01

실시간 수집 및 노이즈 필터링

IoT 센서에서 발생하는 실시간 데이터 스트림을 안정적으로 수집하고, 공학적 도메인 규칙에 기반해 비정상 노이즈를 1차 필터링합니다.

02

견고한 ETL 파이프라인

분석 불가능한 거친 상태의 대용량 데이터를 추출(Extract), 변환(Transform), 적재(Load)하여 클라우드 데이터 웨어하우스에 안정적으로 자동 정제합니다.

03

AI 기반 수율 예측 및 MLOps

정제된 데이터를 머신러닝 모델에 피딩하여 공정 불량률을 실시간 예측하고, 대시보드 및 알림을 통해 현장의 수율 개선 의사결정을 가속화합니다.

Core Strengths & Proof

수율 개선 중심의 엔지니어링

단순히 오류 없는 코드를 작성하는 단계를 넘어, 공정의 병목을 분석하고 파이프라인을 최적화하여 실제 생산 효율 향상을 이루는 것을 목적으로 삼습니다.
